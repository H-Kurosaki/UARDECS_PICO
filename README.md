# UARDECS_PICO
UARDECS is a library for implementing the "UECS" protocol, a Japanese communication standard for horticulture, on Raspberry Pi Pico. To use this library, you need to connect the W5500 as an Ethernet controller. In addition, the Ethernet3 library must be installed with this library.

UARDECSは日本の施設園芸用の通信規格である"UECS"準拠のプロトコルをRaspberry Pi Picoに実装するためのライブラリです。このライブラリを利用するにはイーサネットコントローラとしてW5500を接続する必要があります。さらにこのライブラリを使うときは一緒にEthernet3ライブラリをインストールして下さい。

詳細なマニュアルは準備中です。
